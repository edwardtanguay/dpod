## Run Globally

Go to the dektop and install globally

```bash
  npm install dpod -g
```

start cli with command

```bash
  dpod
```

change to project directory

```bash
  cd <project-name>
  cd <frontend> or <backend>
```

and start the project

```bash
  npm run dev
```
